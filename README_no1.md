# 仕様(1)

## 実行手順
コマンドプロンプト上で，

$ gcc no1.c

を実行し，正常にコンパイルが通ったことを確認したら，

$ ./a.out

を実行する．端末上に

$ オムライス

と出力されればOK．

## 文字化けする場合

nkfコマンド(参考URL:http://itpro.nikkeibp.co.jp/article/COLUMN/20060227/230849/)を使用し，

$ nkf -w --overwrite no1.c

など，適宜実行したいオプションを付与して再度行って下さい．

### 文字コードを確認する方法
fileコマンドを用いる．

$ file --mime [filename]

このようにすると文字コードが確認できる．
